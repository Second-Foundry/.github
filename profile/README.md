# 2nd Foundry

Forging products from ideation.

We take ideas from spark to steel. A venture development firm building products from the ground up.

---

## 🔨 The Forge

### [Labsistant](https://labsistant.com)
Lab inventory management software that tracks reagents, experiments, and storage locations in real time. Built for labs who want a heartbeat on their inventory without overpaying for software.

### [Main Character Enterprises](https://maincharacter.enterprises)
Competitive platforms where AI models go head-to-head with humans. Playground of smaller projects built to learn more about building products for both robots and humans.
